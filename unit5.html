UNIT 5.1: INSERTING IMAGES

HTML IMAGES

The HTML <img>tag is used to embed an image in a web page. The <img> tag is empty, it contains attributes only, and does not have a closing tag. The <img> tag has two required attributes: src and alt.

HTML IMAGE ATTRIBUTES

  • src: specifies the path to the image.
  • alt: specifies an alternate text for the image, if the image for some reason cannot be displayed.
  • title: specifies extra information about an element. The information is most often shown as a tooltip text when the mouse moves over the element.

TYPES OF WEB PAGE IMAGES

  • GIF: Graphics Interchange Format. It is a bitmap image format that supports both animated and static images.
  • JPEG: Joint Photographic Experts Group. It is a commonly used method of lossy compression for digital images.
  • PNG: Portable Network Graphics. It is a raster graphics file format that supports lossless data compression.
  • SVG:Scalable Vector Graphics. It is an XML-based vector image format for two-dimensional graphics with support for interactivity and animation.

NDERSTANDING ABSOLUTE RELATIVE IMAGE URLS

  • Absolute URL: An absolute URL is a complete URL that includes the protocol (http or https), domain name, and path to the image file.
  • Relative URL: A relative URL is a partial URL that does not include the protocol or domain name. It is relative to the current page's location.

IMAGES PLACEMENT AND STYLING

width and height

The width and height properties are used to set the width and height of an image.The values can be set in pixels or as a percentage of the parent element.

max-width

The max-width property is used to set the maximum width of an image. The value can be set in pixels or as a percentage of the parent element.

display and margin

The display property is used to set the display type of an image. The margin property is used to set the margin around an image. The values can be set in pixels or as a percentage of the parent element.

float

The float property is used to set the float type of an image. The value can be set to left, right, or none.

CSS Background properties

Background-image

The Background-image property specifies an image to use as the Background of an element. By default, the image is repeated so it covers the entire element.

Background-repeat

The Background-repeat property repeats an image both horizontally and vertically. The value can be horizontally repeated (repeat-x), vertically repeated (repeat-y), or no-repeat.

Background-attachment

The Background-attachment property specifies whether the background image should scroll or be fixed.

Background-position

The Background-position property specifies the position of the background image. The value can be set in pixels or as a percentage.

Background-size

The CSS background-size property allows you to specify the size of background images. The size can be specified in lenghts, percentage, or by using one of the two keywords: contain or cover.

Background

The CSS background property sets all the background properties in one declaration. When using the shorthand property the order of the property values is:

  • background-color
  • background-image
  • background-repeat
  • background-position
  • background-attachment

HTML Favicon

A favicon is a small image displayed next to the page title in the browser tab. To add a favicon to your website, use the link/ element within the head section of your HTML document. The link/ tag defines the relationship between the current document and an external resource. The link/ element is an empty element; it contains attributes to perform its function.

link/attributes used to create a favicon

  • rel: specifies the relationship between the current document and the linked resource.
  • href: specifies the URL of the linked resource.
  • type: specifies the media type of the linked resource.
